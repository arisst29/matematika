Kvadratinės, aukštesniojo laipsnio ir trupmeninės nelygybės

Kvadratines ir aukštesniojo laipsnio nelygybes sprendžiame intervalų metodu — universaliu būdu, tinkančiu daugeliui nelygybių tipų.

1
Intervalų metodas
Metodo esmė
Pirmiausia išsprendžiame pradinei nelygybei analogišką lygtį. Jos sprendiniai padalija skaičių tiesę į intervalus. Kiekviename intervale nustatome nelygybės reiškinio ženklą ir parenkame tinkamą atsakymą.
4 žingsnių algoritmas
  1. Visus nelygybės narius perkeliame į kairiąją pusę ir išskaidome daugikliais.
  2. Nelygybės ženklą pakeičiame lygybės ženklu ir išsprendžiame lygtį — randame x reikšmes, su kuriomis kiekvienas daugiklis lygus nuliui. Šios reikšmės padalija skaičių tiesę į intervalus.
  3. Kiekviename intervale nustatome daugiklių ženklus — įrašome vieną to intervalo x reikšmę ir sužinome, kokią reikšmę (teigiamą ar neigiamą) kiekviename intervale įgyja nelygybės reiškinio daugiklis. Jei neigiamų daugiklių skaičius lyginis — sandauga teigiama, jei nelyginis — neigiama.
  4. Užrašome nelygybės sprendinius — t. y. tuos x reikšmių intervalus, kuriuose reiškinio ženklas sutampa su nelygybės ženklu. Atkreipiame dėmesį į taškus, kuriuose reiškinys lygus nuliui.
Taškai skaičių tiesėje
Taškai, kuriuose reiškinys lygus nuliui:
• Jei nelygybė griežta (< arba >) — taškai neįeina į sprendinį (žymima tuščiaviduriu skrituliu ○)
• Jei nelygybė negriežta (≤ arba ≥) — taškai įeina į sprendinį (žymima pilnaviduriu skrituliu ●)
Pavyzdys — kvadratinė nelygybė
Išspręskime nelygybę x² − x − 6 ≤ 0.

Išsprendžiame lygtį x² − x − 6 = 0: D = 1 + 24 = 25; x₁ = −2, x₂ = 3 Išskaidome: x² − x − 6 = (x + 2)(x − 3).
Pažymime x = −2 ir x = 3 skaičių tiesėje — ji dalijama į tris intervalus.
Iš kiekvieno intervalo paimame po vieną skaičių ir tikriname ženklą:
x = −3: (−1)(−6) = +   x = 0: (2)(−3) = −   x = 5: (7)(2) = +
Nelygybė ≤ 0, tai imame intervalą su minusu: [−2; 3]. Atsakymas: x ∈ [−2; 3]

SKAIČIŲ TIESĖ:
● — taškas įeina (≤), sprendinys: x ∈ [−2; 3]
Pavyzdys — aukštesniojo laipsnio nelygybė
Išspręskime nelygybę (x + 3)(x − 1)(x − 4) > 0.

Daugikliai lygūs nuliui kai x = −3, x = 1, x = 4. Tiesė dalijama į 4 intervalus.
Tikriname ženklą kiekviename intervale:
x = −4: (−)(−)(−) = −   x = 0: (+)(−)(−) = +   x = 2: (+)(+)(−) = −   x = 5: (+)(+)(+) = +
Nelygybė > 0, imame teigiamus intervalus (taškai neįeina): Atsakymas: x ∈ (−3; 1) ∪ (4; +∞)

SKAIČIŲ TIESĖ:
○ — taškas neįeina (>), sprendinys: x ∈ (−3; 1) ∪ (4; +∞)
2
Trupmeninės nelygybės
Skirtumas nuo įprastų nelygybių
Trupmeninės nelygybės sprendžiamos intervalų metodu panašiai kaip įprastos, tačiau yra vienas svarbus skirtumas — taškus, kuriuose vardiklis lygus nuliui, reikia pažymėti tuščiaviduriais skrituliukais. Šie taškai nėra nelygybės sprendiniai.
Sprendimo tvarka
  1. Nelygybę pertvarkome taip, kad ji įgytų pavidalą A(x)/B(x) > 0 (vietoj ženklo „>" gali būti ženklas „<", „≤" arba „≥").
  2. Sprendžiame lygtį A(x) = 0 ir nelygybę B(x) ≠ 0. Gautus taškus pažymime skaičių tiesėje ir padalijame ją į intervalus.
  3. Nustatome reiškinio ženklą kiekviename intervale, atsižvelgdami į sąlygos reikalavimus bei apibrėžimo sritį, užrašome atsakymą.
Pavyzdys
Išspręskime nelygybę (x + 4) / (x − 2) ≤ 0.

Skaitiklis lygus nuliui kai x = −4, vardiklis lygus nuliui kai x = 2.
Pažymime skaičių tiesėje: x = −4 (pilnaviduris ●, nes ≤), x = 2 (tuščiaviduris ○, nes vardiklis).
Tikriname ženklą intervaluose:
x = −5: (−)/(−) = +   x = 0: (+)/(−) = −   x = 3: (+)/(+) = +
Imame intervalą su minusu ir tašką x = −4: Atsakymas: x ∈ [−4; 2)

SKAIČIŲ TIESĖ:
Sprendinys: x ∈ [−4; 2)
Svarbu
Vardiklio nuliniai taškai niekada nepriklauso trupmeninės nelygybės sprendiniams — net kai nelygybė negriežta (≤ arba ≥). Todėl šie taškai visada žymimi tuščiaviduriu skrituliuku ○.
3
Kvadratinės lygties sprendinių skaičiaus nustatymas
Diskriminanto taisyklė
Kvadratinė lygtis ax² + bx + c = 0 (a ≠ 0):

• turi du skirtingus realiuosius sprendinius, kai D = b² − 4ac > 0
• turi vieną kartotinį sprendinį, kai D = 0
• neturi realiųjų sprendinių, kai D < 0

Tai naudojama sudarant nelygybes, kad nustatytume, kokioms parametro reikšmėms lygtis turi tam tikrą sprendinių skaičių.
Pavyzdys
Su kuriomis m reikšmėmis lygtis x² + (m − 1)x + m = 0 turi du skirtingus realiuosius sprendinius?

Sudarome nelygybę D > 0: (m − 1)² − 4m > 0 m² − 2m + 1 − 4m > 0 m² − 6m + 1 > 0 Sprendžiame intervalų metodu: m₁ = 3 − 2√2, m₂ = 3 + 2√2. Atsakymas: m ∈ (−∞; 3 − 2√2) ∪ (3 + 2√2; +∞)
SKAIČIŲ TIESĖ (parametras m):
○ — taškai neįeina (>), sprendinys: m ∈ (−∞; 3−2√2) ∪ (3+2√2; +∞)
