NelygybiЕі sistemos, dvigubosios nelygybД—s ir nelygybД—s su moduliu

NelygybiЕі sistemЕі sprendiniai вҖ” bendrieji visЕі sistemos nelygybiЕі sprendiniai. Dvigubosios nelygybД—s ir nelygybД—s su moduliu sprendЕҫiamos pertvarkant ДҜ sistemas.

1
NelygybiЕі sistemos
Sprendimo tvarka
  1. Randame kiekvienos sistemos nelygybД—s sprendiniЕі aibДҷ atskirai.
  2. Visas sprendiniЕі aibes pavaizduojame toje paДҚioje skaiДҚiЕі tiesД—je вҖ” vienД… aibДҷ brЕ«kЕЎniuoti virЕЎuje, kitД… вҖ” apaДҚioje.
  3. Randame pavaizduotЕі aibiЕі sankirtД… вҖ” tai ir yra sistemos sprendiniai. Jei aibiЕі bendrЕі sprendiniЕі nД—ra вҖ” sistema neturi sprendiniЕі.
Pavyzdys
IЕЎsprДҷskime nelygybiЕі sistemД…: { 2x вҲ’ 6 > 0; вҲ’xВІ + 2x + 8 вүҘ 0 }

1 nelygybД—: 2x вҲ’ 6 > 0 вҶ’ x > 3 вҶ’ x вҲҲ (3; +вҲһ)

2 nelygybД—: вҲ’xВІ + 2x + 8 вүҘ 0, dauginame iЕЎ (вҲ’1): xВІ вҲ’ 2x вҲ’ 8 вүӨ 0 вҶ’ (x вҲ’ 4)(x + 2) вүӨ 0 IntervalЕі metodu: x вҲҲ [вҲ’2; 4]

Randame sankirtД…: (3; +вҲһ) вҲ© [вҲ’2; 4] Atsakymas: x вҲҲ (3; 4]

2
Dvigubosios nelygybД—s
ApibrД—Еҫimas
DvigubД…jД… nelygybДҷ a < f(x) < b visada galima uЕҫraЕЎyti kaip nelygybiЕі sistemД…: { f(x) > a ir f(x) < b } TaДҚiau, jeigu f(x) yra pirmojo laipsnio daugianaris, sprДҷsti galima ir tiesiogiai вҖ” atliekant veiksmus su visomis trimis dalimis.
Pavyzdys
IЕЎsprДҷskime nelygybДҷ вҲ’4 < 3 вҲ’ 2x вүӨ 10.

IЕЎ visЕі trijЕі daliЕі atimame 3: вҲ’4 вҲ’ 3 < вҲ’2x вүӨ 10 вҲ’ 3 вҲ’7 < вҲ’2x вүӨ 7 Dalijame iЕЎ (вҲ’2) вҖ” Еҫenklai keiДҚiasi: 7/2 > x вүҘ вҲ’7/2 Atsakymas: x вҲҲ [вҲ’3,5; 3,5)
SKAIДҢIЕІ TIESД–:
Sprendinys: x вҲҲ [вҲ’3,5; 3,5)
3
NelygybД—s su moduliu
PagrindinД—s taisyklД—s (a > 0)
NelygybД—Ekvivalenti forma
|f(x)| < aвҲ’a < f(x) < a   (dviguboji nelygybД—)
|f(x)| вүӨ aвҲ’a вүӨ f(x) вүӨ a
|f(x)| > af(x) < вҲ’a arba f(x) > a
|f(x)| вүҘ af(x) вүӨ вҲ’a arba f(x) вүҘ a

Kai a < 0: |f(x)| < a вҖ” sprendiniЕі nД—ra; |f(x)| > a вҖ” visi x iЕЎ apibrД—Еҫimo srities.
Pavyzdys 1 вҖ” |f(x)| < a
IЕЎsprДҷskime nelygybДҷ |4x вҲ’ 2| < 10.

Pertvarkome ДҜ dvigubД…jД… nelygybДҷ: вҲ’10 < 4x вҲ’ 2 < 10 Pridedame 2 visur: вҲ’8 < 4x < 12 Dalijame iЕЎ 4: вҲ’2 < x < 3 Atsakymas: x вҲҲ (вҲ’2; 3)
SKAIДҢIЕІ TIESД–:
в—Ӣ вҖ” neДҜeina (<), sprendinys: x вҲҲ (вҲ’2; 3)
Pavyzdys 2 вҖ” |f(x)| > a
IЕЎsprДҷskime nelygybДҷ |2x + 6| > 4.

SprendЕҫiame dvi nelygybes: 2x + 6 < вҲ’4 вҶ’ 2x < вҲ’10 вҶ’ x < вҲ’5 2x + 6 > 4 вҶ’ 2x > вҲ’2 вҶ’ x > вҲ’1 Atsakymas: x вҲҲ (вҲ’вҲһ; вҲ’5) вҲӘ (вҲ’1; +вҲһ)
SKAIДҢIЕІ TIESД–:
в—Ӣ вҖ” neДҜeina (>), sprendinys: x вҲҲ (вҲ’вҲһ; вҲ’5) вҲӘ (вҲ’1; +вҲһ)

ЕҪodiniЕі uЕҫdaviniЕі sprendimas sudarant nelygybes
Algoritmas
AnalogiЕЎkai kaip sudarant lygtis, tik vietoj lygybД—s sudarome nelygybДҷ pagal uЕҫdavinio sД…lygoje esanДҚius ЕҫodЕҫius: вҖһdaugiau", вҖһmaЕҫiau", вҖһne maЕҫiau", вҖһne daugiau", вҖһbent", вҖһdaugiausia" ir t. t.
Pavyzdys
PardavД—jui siЕ«lomi du atlyginimЕі planai: A вҖ” fiksuotas 500 eurЕі per mД—nesДҜ, arba B вҖ” 200 eurЕі ir 8% komisiniai nuo pardavimЕі sumos. UЕҫ kokiД… sumД… reikia parduoti, kad planas B bЕ«tЕі naudingesnis?

Tarkime, pardavimЕі suma x eurЕі. Planas B naudingesnis, kai: 200 + 0,08x > 500 0,08x > 300 x > 3750 Atsakymas: reikia parduoti daugiau nei uЕҫ 3750 eurЕі.
