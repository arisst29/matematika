Reiškinio apibrėžimo sritis ir veiksmai su reiškiniais

Sužinosime, kada algebrinis reiškinys turi prasmę ir kaip suprastinti sudėtingus reiškinius su šaknimis bei laipsniais.

1
Reiškinio apibrėžimo sritis (T4)
Apibrėžimas
Algebrinio reiškinio apibrėžimo sritimi vadinsime kintamojo reikšmių, su kuriomis šis reiškinys turi prasmę, aibę.

Apibrėžimo sritis žymima raide D.
5 pagrindinės taisyklės
#ReiškinysSąlyga
1Sveikasis reiškinysx ∈ ℝ (visada apibrėžtas)
2Trupmeninis reiškinys g(x)/f(x)f(x) ≠ 0 (vardiklis ≠ 0)
3Lyginio laipsnio šaknis skaitiklyje ²ⁿ√f(x)f(x) ≥ 0
4Lyginio laipsnio šaknis vardiklyje g(x)/²ⁿ√f(x)f(x) > 0 (nes ir ≠ 0, ir ≥ 0)
5Laipsnis su racionaliu rodikliu aᵐ/ⁿa > 0
Pavyzdys 1 — sveikasis reiškinys
Nustatykime reiškinio 2x³ − 5x + √7 apibrėžimo sritį.

Reiškinys yra sveikasis (nėra nei trupmenų, nei lyginių šaknų su kintamuoju pošaknyje), todėl pagal 1 taisyklę apibrėžtas visiems x ∈ ℝ. D: x ∈ ℝ
Pavyzdys 2 — trupmeninis reiškinys
Nustatykime reiškinio (x + 3) / (x − 7) apibrėžimo sritį.

Pagal 2 taisyklę vardiklis negali būti lygus nuliui: x − 7 ≠ 0 x ≠ 7 D: x ∈ ℝ \ {7} Arba: x ∈ (−∞; 7) ∪ (7; +∞)
Pavyzdys 3 — šaknis ir trupmenė
Nustatykime reiškinio √(3 + x) / (x − 4) apibrėžimo sritį.

Pagal 3 taisyklę pošaknis turi būti neneigiamas, pagal 2 — vardiklis nelygus nuliui: { 3 + x ≥ 0 → x ≥ −3 { x − 4 ≠ 0 → x ≠ 4 Abu apribojimai turi galioti vienu metu: D: x ∈ [−3; 4) ∪ (4; +∞)
Svarbu
Kai šaknis yra lyginio laipsnio (√, ⁴√, ...) ir yra vardiklyje, pošaknis turi būti teigiamas (> 0, o ne ≥ 0), nes nuliu dalinti negalima.

Kai šaknis yra skaitiklyje, pošaknis turi būti neneigiamas (≥ 0).
2
Veiksmai su algebriniais reiškiniais
Daugiklio įkėlimas po šaknies ženklu
Jei nurodyta, kad kintamojo reikšmės yra teigiamos, arba šaknis yra nelyginio laipsnio, daugiklį galima įkelti po šaknies ženklu:

Kai x > 0: x · √y = √(x²y) Jei sąlyga nenurodyta — naudokite modulį: x · √y = √(x²y), kai x ≥ 0; −√(x²y), kai x < 0.
Pavyzdys — daugiklio įkėlimas
Suprastinkime reiškinį ⁴√(x²y²) · √(xy), kai x > 0 ir y > 0.

Sulyginame šaknų rodiklius — t. y. po trečiojo laipsnio šaknimi esančius dauginamus keliame kvadratu: ⁴√(x²y²) · √(xy) = ⁴√(x²y²) · ⁴√(x²y²) = ⁴√(x⁴y⁴) = x · y = xy
Lyginio laipsnio šaknis ir modulis
Traukiant lyginio laipsnio šaknį iš reiškinio, pakeltą lyginiu laipsniu, visada gaunamas neneigiamas skaičius: ²ⁿ√(a²ⁿ) = |a| Jei pošakninis reiškinys yra teigiamas — modulio ženklo nereikia. Jei neigiamas — reikia:

Pavyzdžiui: ⁴√(x − 4)⁴ + ³√(x − 2)³, kai x < 4
= |x − 4| + (x − 2) Kai x < 4: reiškinys x − 4 < 0, todėl |x − 4| = −(x − 4) = 4 − x: = (4 − x) + (x − 2) = 2
Pavyzdys — suprastinimas su laipsniais
Suprastinkime reiškinį x^(2/5) · (x^(3/5) + x^(−2/5)).

Atskliaučiame — dauginame laipsnį iš laipsnio, rodiklius sudedame: = x^(2/5) · x^(3/5) + x^(2/5) · x^(−2/5) = x^(2/5 + 3/5) + x^(2/5 − 2/5) = x¹ + x⁰ = x + 1
