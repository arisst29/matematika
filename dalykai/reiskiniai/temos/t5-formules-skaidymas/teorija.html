Greitosios daugybos formulės, skaidymas daugikliais ir algebrinės trupmenos

Trys glaudžiai susijusios temos — formulės leidžia greičiau dauginti, skaidymas daugikliais padeda suprastinti, o su trupmenomis atliekami visi keturi veiksmai.

1
Greitosios daugybos formulės (T5)
7 pagrindinės formulės
① Kvadrato suma
(a + b)² = a² + 2ab + b²
② Kvadrato skirtumas
(a − b)² = a² − 2ab + b²
③ Kvadratų skirtumas
a² − b² = (a − b)(a + b)
④ Kubo suma
(a + b)³ = a³ + 3a²b + 3ab² + b³
⑤ Kubo skirtumas
(a − b)³ = a³ − 3a²b + 3ab² − b³
⑥ Kubų suma
a³ + b³ = (a + b)(a² − ab + b²)
⑦ Kubų skirtumas
a³ − b³ = (a − b)(a² + ab + b²)
Pavyzdys — pakelimas laipsniu
Pakelkime reiškinį (3x + 2) kubu.

Taikome 4 formulę, kur a = 3x, b = 2: (3x + 2)³ = (3x)³ + 3·(3x)²·2 + 3·(3x)·2² + 2³ = 27x³ + 54x² + 36x + 8
Pavyzdys — išskaidymas daugikliais pagal formulę
Išskaidykime 25x⁴y² − 9b⁶ daugikliais.

Kiekvieną dvinario narį parašome kaip kvadratą: 25x⁴y² = (5x²y)², 9b⁶ = (3b³)². Taikome 3 formulę: 25x⁴y² − 9b⁶ = (5x²y − 3b³)(5x²y + 3b³)
2
Skaidymas daugikliais
4 pagrindiniai būdai
1. Kėlimas prieš skliaustus — bendras daugiklis iškeliamas:
ab ± ac = a(b ± c) 2. Greitosios daugybos formulių taikymas — atpažįstame formulę ir skaidome.

3. Kvadratinio trinario skaidymas:
ax² + bx + c = a(x − x₁)(x − x₂) čia x₁ ir x₂ — lygties ax² + bx + c = 0 sprendiniai (jei D ≥ 0).

4. Grupavimas — narius sugrupuojame taip, kad iš kiekvienų galima iškelti bendrą daugiklį.
Pavyzdys — kėlimas prieš skliaustus
Iškelkime daugiklį prieš skliaustus: x − 2x√x + √x

Pastebime, kad bendrasis daugiklis yra √x. Galime rašyti x = (√x)²: x − 2x√x + √x = √x · √x − 2x · √x + √x · 1 = √x(√x − 2x + 1)
Pavyzdys — kvadratinio trinario skaidymas
Išskaidykime daugikliais 6x² + x − 2.

Pirma išsprendžiame kvadratinę lygtį 6x² + x − 2 = 0:
D = 1 + 48 = 49, x₁ = 1/2, x₂ = −2/3

Išskaidome pagal formulę: 6x² + x − 2 = 6(x − 1/2)(x + 2/3) = (2x − 1)(3x + 2)
3
Algebrinės trupmenos
Suprastinimas
Algebrinę trupmeną suprastiname išskaidydami skaitiklį ir vardiklį daugikliais ir sutraukdami bendrus daugiklius: (ac) / (bc) = a/b, kai c ≠ 0 Svarbu: sutraukti galima tik daugiklius, o ne narius!
Pavyzdys — suprastinimas
Suprastinkime (9a² − 9b²) / (4b − 4a).

Išskaidome skaitiklį ir vardiklį: 9a² − 9b² = 9(a² − b²) = 9(a − b)(a + b) 4b − 4a = 4(b − a) = −4(a − b) Sutraukiame bendrą daugiklį (a − b): = 9(a − b)(a + b) / [−4(a − b)] = −9(a + b) / 4
Sudėtis ir atimtis
Sudedant trupmenas reikia:
1. Išskaidyti vardiklius daugikliais
2. Nustatyti bendrąjį vardiklį
3. Rasti kiekvienai trupmenai papildomąjį daugiklį
4. Užrašyti reiškinį viena trupmena ir suprastinti

a/b + c/d = (ad + bc) / bd a/b − c/d = (ad − bc) / bd
Pavyzdys — sudėtis
Suprastinkime −3p/(2p − 8) − (p + 2)/(4 − p).

Pirmosios trupmenos vardiklį išskaidome: 2p − 8 = 2(p − 4).
Antrosios vardiklyje esančio dvinario nariai sukeičiami vietomis: 4 − p = −(p − 4).
Bendrasis vardiklis yra 2(p − 4): = −3p / [2(p−4)] + (p+2) / (p−4) = [−3p + 2(p+2)] / [2(p−4)] = (−3p + 2p + 4) / [2(p−4)] = (−p + 4) / [2(p−4)] = −(p − 4) / [2(p−4)] = −1/2
Daugyba ir dalyba
Algebrinės trupmenos dauginamos: (a/b) · (c/d) = (a·c) / (b·d) Dalijant — dalijamąją trupmeną dauginame iš dalytojo atvirkštinės: (a/b) ÷ (c/d) = (a/b) · (d/c) = (a·d) / (b·c) Veiksmai atliekami tokia eilės tvarka: pirmiausia daugyba ir dalyba, paskui sudėtis ir atimtis.
Pavyzdys — daugyba ir dalyba
Padalinkime (x⁴ − y⁴) / (x² + 2xy + y²) ÷ (x² + y²) / (x + y).

Keičiame dalybą daugyba su atvirkštine trupmena: = (x⁴ − y⁴) / (x + y)² · (x + y) / (x² + y²) Išskaidome x⁴ − y⁴ = (x² − y²)(x² + y²) = (x−y)(x+y)(x²+y²): = (x−y)(x+y)(x²+y²) / [(x+y)² · (x²+y²)] · (x+y) = x − y
Svarbu — dažnos klaidos
❌ Negalima sutraukti narius: (a + c) / (b + c) ≠ a/b
✓ Galima sutraukti tik daugiklius: (a · c) / (b · c) = a/b

❌ Negalima sudėti skirtingų vardiklių skaitiklių: a/b + c/d ≠ (a+c)/(b+d)
✓ Reikia bendro vardiklio
