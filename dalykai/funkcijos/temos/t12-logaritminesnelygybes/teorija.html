Logaritminės nelygybės

Nelygybės su logaritmais — ženklo keitimas priklauso nuo pagrindo.

1
Pagrindinis principas — ženklo keitimas
Svarbu
Logaritminėje nelygybėje logₐf(x) > logₐg(x) sprendimas priklauso nuo to, ar pagrindas a didesnis ar mažesnis už 1. Tai lemia ar ženklas keičiasi!
Taisyklė
Kai a > 1 (didėjanti)

logₐf(x) > logₐg(x)

⟺ f(x) > g(x)

ir f(x) > 0, g(x) > 0

Ženklas nesikeičia

Kai 0 < a < 1 (mažėjanti)

logₐf(x) > logₐg(x)

⟺ f(x) < g(x)

ir f(x) > 0, g(x) > 0

Ženklas keičiasi į priešingą!

Pavyzdys — a > 1
log₄(x − 2) > 3

Skaičių 3 užrašome kaip logaritmą: 3 = log₄4³ = log₄64
Nelygybių sistema (pagrindas 4 > 1, ženklas nesikeičia):
• x − 2 > 64 → x > 66
• x − 2 > 0 → x > 2

Atsakymas: x ∈ (66; +∞)
Pavyzdys — 0 < a < 1
log₍₁/₄₎(x − 2) > 3

3 = log₍₁/₄₎(1/4)³ = log₍₁/₄₎(1/64)
Pagrindas 1/4 < 1, ženklas keičiasi:
• x − 2 < 1/64 → x < 2 + 1/64
• x − 2 > 0 → x > 2

Atsakymas: x ∈ (2; 2 + 1/64)
2
Sudėtingesnių nelygybių sprendimas
Būdo esmė
Sudėtingesnės logaritminės nelygybės sprendžiamos panašiai kaip logaritminės lygtys — taikant logaritmų savybes, pakeičiant į paprastesnę nelygybę.
Pavyzdys — su logaritmų savybėmis
log₂(x−1) − log₂5 ≤ 1 + log₂(12−x)

1 = log₂2, todėl:
log₂(x−1) − log₂5 ≤ log₂2 + log₂(12−x)
log₂((x−1)/5) ≤ log₂(2(12−x))

Pagrindas 2 > 1, ženklas nesikeičia:
(x−1)/5 ≤ 2(12−x)
x − 1 ≤ 10(12 − x)
x − 1 ≤ 120 − 10x
11x ≤ 121 → x ≤ 11

Pradinės lygties sąlygos: x − 1 > 0 ir 12 − x > 0 → 1 < x < 12
Atsakymas: x ∈ (1; 11]
3
Keitinio metodas nelygybėms
Būdo esmė
Sudėtingesnėse nelygybėse naudojame keitinį t = logₐx, gauname kvadratinę nelygybę ir išsprendžiame ją. Atkreipiame dėmesį į pagrindą — ar keičiasi ženklas.
Pavyzdys
7log²₅x − 4log₅x − 3 > 0

Keitinys t = log₅x:
7t² − 4t − 3 > 0
7(t − 1)(t + 3/7) > 0
t < −3/7 arba t > 1

• log₅x < −3/7 → 0 < x < 5^(−3/7)
• log₅x > 1 → x > 5

Atsakymas: x ∈ (0; 5^(−3/7)) ∪ (5; +∞)
