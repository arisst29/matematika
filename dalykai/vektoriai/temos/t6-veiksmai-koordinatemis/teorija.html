Veiksmai su vektoriais koordinatėmis

Žinodami vektorių koordinates, galime lengvai atlikti sudėtį, atimtį ir daugybą iš skaičiaus — tiesiog veikiame su atitinkamomis koordinatėmis atskirai.

1
Sudėtis, atimtis ir daugyba iš skaičiaus
Taisyklė
Jei a→ = (xa; ya) ir b→ = (xb; yb), k — bet koks realusis skaičius, tai:

Sudėtis: a→ + b→ = (xa + xb;  ya + yb) Atimtis: a→ − b→ = (xa − xb;  ya − yb) Daugyba iš skaičiaus: ka→ = (k·xa;  k·ya) Kitaip tariant: veikiame su kiekviena koordinate atskirai.
Pavyzdys
a→ = (4; −2), b→ = (1; 3). Apskaičiuokite 3a→ − 2b→ ir a→ + b→.

3a→ − 2b→: = (3·4 − 2·1;  3·(−2) − 2·3) = (12−2;  −6−6) = (10;  −12) a→ + b→: = (4+1;  −2+3) = (5;  1)
2
Kolinearumo sąlyga koordinatėmis
Teorema
Nenuliniai vektoriai a→ = (xa; ya) ir b→ = (xb; yb) yra kolinearūs tada ir tik tada, kai jų koordinatės yra proporcingos: xa / xb = ya / yb  (arba lygiavertiškai: xa · yb − xb · ya = 0) Jei k > 0 — vektoriai vienakrypčiai.
Jei k < 0 — vektoriai priešpriešiniai.
Svarbu
Ši proporcijų formulė netaikoma vektoriams, kurie yra lygiagrečiai koordinačių ašims — tada viena koordinatė lygi nuliui ir dalyba neapibrėžta.

Pavyzdys: a→ = (4; 0) ir b→ = (−6; 0). Abu vektoriai lygiagrečiai ašiai Ox, todėl ya = yb = 0 — negalime dalinti. Naudojame kryžminę sandaugą: 4 · 0 − (−6) · 0 = 0 ✓  → vektoriai kolinearūs Tokiu atveju naudokite: xa · yb − xb · ya = 0
Pavyzdys
Žinoma, kad u→ = 4i→ + 2j→, v→ = 1i→ + 4j→, t→ = (2; 7), z→ = (8; 17). Ar a→ = u→ − v→ ir b→ = t→ − z→ yra kolinearūs?

a→ = (4−1; 2−(−3)) = (3; 5)
b→ = (2−8; 7−17) = (−6; −10)

Tikriname: 3 / (−6) = −½,  5 / (−10) = −½ Santykiai lygūs, taigi vektoriai kolinearūs (ir priešpriešiniai, nes k = −½ < 0).
3
Taškų kolinearumas tiesėje
Taisyklė
Taškai A, B, C yra vienoje tiesėje tada ir tik tada, kai vektoriai AB→ ir AC→ yra kolinearūs, t. y.: xAB · yAC − xAC · yAB = 0
Pavyzdys
Įrodykite, kad taškai A(0; −3), B(3; 3) ir C(−2; −7) yra vienoje tiesėje.

AB→ = (3−0; 3−(−3)) = (3; 6)
AC→ = (−2−0; −7−(−3)) = (−2; −4)

Tikriname kolinearumą: 3 · (−4) − (−2) · 6 = −12 + 12 = 0 ✓ Kadangi rezultatas lygus nuliui, vektoriai kolinearūs — taškai A, B, C yra vienoje tiesėje.
