Skaliarinė sandauga ir kampas koordinatėmis

Žinodami vektorių koordinates, galime apskaičiuoti skaliarinę sandaugą paprastu veiksmu — be kampų. O iš skaliarinės sandaugos lengvai randame kampą tarp vektorių.

1
Skaliarinė sandauga koordinatėmis
Formulė
Jei a→ = (xa; ya) ir b→ = (xb; yb), tai: a→ · b→ = xa · xb + ya · yb Skaliarinė sandauga lygi atitinkamų koordinačių sandaugų sumai.

Taip pat galioja: |a→|² = a→ · a→ = xa² + ya²
Pavyzdys
a→ = (4; −2), b→ = (1; 3). Apskaičiuokite (3a→ − 2b→) · (a→ + b→).

1. Apskaičiuojame 3a→ − 2b→: = (3·4 − 2·1;  3·(−2) − 2·3) = (10;  −12) 2. Apskaičiuojame a→ + b→: = (4+1;  −2+3) = (5;  1) 3. Skaliarinė sandauga: (10; −12) · (5; 1) = 10·5 + (−12)·1 = 50 − 12 = 38 Atsakymas: 38.
2
Statmenumo sąlyga koordinatėmis
Teorema
Nenuliniai vektoriai a→ = (xa; ya) ir b→ = (xb; yb) yra statmeni tada ir tik tada, kai: xa · xb + ya · yb = 0 Tai tiesiogiai išplaukia iš T4 — kai φ = 90°, cos 90° = 0, todėl sandauga = 0.
Pavyzdys
Ar vektoriai AB→ ir CD→ yra statmeni, jei A(1; 1), B(3; 7), C(0; 2), D(6; 0)?

AB→ = (3−1; 7−1) = (2; 6)
CD→ = (6−0; 0−2) = (6; −2)

Tikriname: 2·6 + 6·(−2) = 12 − 12 = 0 ✓ Taip, AB→ ⊥ CD→.
3
Kampas tarp vektorių koordinatėmis
Išvedimas
Pagrindinė skaliarinės sandaugos formulė (iš T4): a→ · b→ = |a→| · |b→| · cos φ Bet koordinatėmis žinome, kad a→ · b→ = xa·xb + ya·yb. Todėl: |a→| · |b→| · cos φ = xa·xb + ya·yb Išreiškiame cos φ — dalijame abi puses iš |a→|·|b→|: cos φ = (xa·xb + ya·yb) / (|a→| · |b→|) kur  |a→| = √(xa² + ya²),  |b→| = √(xb² + yb²)

Kampas φ ∈ [0°; 180°].
Pavyzdys
Raskite kampą tarp vektorių a→ = (2; −3) ir b→ = (1; 5).

Skaliarinė sandauga: a→ · b→ = 2·1 + (−3)·5 = 2 − 15 = −13 Ilgiai: |a→| = √(4 + 9) = √13 |b→| = √(1 + 25) = √26 Kampas: cos φ = −13 / (√13 · √26) = −13 / (13√2) = −1/√2 ≈ −0,707 φ = arccos(−0,707) ≈ 135° Atsakymas: φ ≈ 135°.
Žingsnis po žingsnio
Norėdami rasti kampą tarp dviejų vektorių koordinatėmis:

1. Apskaičiuokite skaliarinę sandaugą: xa·xb + ya·yb
2. Apskaičiuokite abiejų vektorių ilgius
3. Padalinkite sandaugą iš ilgių sandaugos — gausite cos φ
4. Raskite φ = arccos(...)
